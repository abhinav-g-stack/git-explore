# EcomWave - A Next.js E-commerce Starter

This is a modern e-commerce application built with Next.js, Tailwind CSS, ShadCN UI, and Genkit for AI-powered features. It was prototyped in Firebase Studio.

## Getting Started

To get started with the application, take a look at `src/app/page.tsx`.
