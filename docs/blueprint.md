# **App Name**: EcomWave

## Core Features:

- Dynamic Navbar: Implement a responsive navigation bar with guest and user-specific options (login/logout, theme switch, categories).
- Homepage UI: Design a hero section and category-based product listings with clean, clickable product cards.
- Footer Design: Implement a clean, responsive footer with relevant links and information.
- Login Page: Create a login page with a split-screen design: e-commerce-related image on the left, login form on the right.
- User Modal: Implement a collapsible cart/menu modal for logged-in users with logout and user preference options.
- Shop Page: Develop a shop page with search, category filters, and product cards with 'add to cart' functionality.
- Admin Dashboard: Create an admin dashboard with collapsible menu, user management, product management, and performance insights using charts.

## Style Guidelines:

- Light Mode: Primary color: Light orange (#FFB347) for a friendly and inviting feel.
- Dark Mode: Primary color: Light cyan (#77DDE7) for a modern and clean aesthetic.
- Light Mode: Background color: Very light orange (#FFF3E0) for a soft, warm base.
- Dark Mode: Background color: Dark gray (#212121) provides high contrast.
- Light Mode: Accent color: Slightly desaturated light yellow (#FFE082) complements the orange without overpowering.
- Dark Mode: Accent color: Slightly brighter, saturated light blue (#80CBC4) offers visual pop.
- Body and headline font: 'Inter' sans-serif font. Inter is versatile enough for both headings and body text, ensuring readability and a clean aesthetic.
- Code Font: 'Source Code Pro' for display of data within the application
- Use clean, modern icons from a library like Material Icons to represent categories, cart actions, and user controls.
- Employ a responsive grid layout with Tailwind CSS to ensure optimal viewing on all devices.
- Incorporate subtle hover effects and transitions to enhance user engagement without being distracting.
